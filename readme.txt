Better Ads is aimed to make managing  ads easy on your wordpress site.

== Description ==

Better Ads makes your ad management as simple as should be. All you need is to provide the code of your ad from admin panel.

It includes a very simple user interface in the admin panel. Paste your ad code in the input and save. And the plugin will manage the rest.
You'll see the ads placed in the middle of your single posts. It's that simple! 


**Upcoming features:**
* Multiple Ad Provider Support
* Improved and nicer Admin panel
* Custom Placement of the Ads; User will be able to decide where they want the ads to show up.


== Installation ==

Upload the plugin to your blog, Activate it. Go To Ad Master Menu, Paste the code in the input box and click Save. 
You're done! The Plugin will handle the rest.
